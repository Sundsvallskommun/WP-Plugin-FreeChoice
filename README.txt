== Description ==

* When the plugin is activated a ws call to the smex endpoint will be triggered, fetching nyko-data and stores it in database
* The nyko data should be updated with a cron job

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload plugin to the `plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the [free-choice] shortcode on any page where you want the plugin to appear
